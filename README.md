# Kickstart Project

Welcome to the Kickstart Project! This project includes various libraries and features to help you get started quickly with your React application.

## Libraries Included

- **WaveSurfer.js**: A library for creating customizable audio waveforms.
- **Chakra UI**: A simple, modular, and accessible component library that provides building blocks to build React applications with ease.
- **Tailwind CSS**: A utility-first CSS framework for creating custom designs without leaving your HTML.
- **React-Redux**: A library that helps you manage state in your React application with Redux.
- **React-Router-dom**: A library that helps you manage routes.

## Additional Features

- **Page Not Found**: A custom page to handle 404 errors when users navigate to a non-existent route.

   ```bash

   cd kickstart
   ```
